ソースからQiskit SDKとQiskit Runtimeをインストールする

ソースからQiskit SDKをインストールすることで、Python Package Index（PyPI）リポジトリのバージョンではなく、現在の開発版にアクセスできます。これにより、Qiskitコードの最新バージョンをより効率的に確認・拡張できます。

新しい仮想環境を作成して有効化する​

  1. プロジェクトディレクトリに移動し、Pythonのみがインストールされた最小限の環境を作成します。

    • macOS
    • Linux
    • Windows
    python3 -m venv .venv
    python3 -m venv .venv
    python -m venv .venv

  2. 新しい環境を有効化します。

    • macOS
    • Linux
    • Windows
    source .venv/bin/activate
    source .venv/bin/activate
    .venv\Scripts\Activate.ps1

Rustコンパイラをインストールする​

Qiskitをコンパイルするには、システムにRustコンパイラがインストールされている必要があります。Rustコンパイラをインストールするには、クロスプラットフォームのRustインストーラーrustup、またはその他のインストール方法を使用してください。

Qiskitをインストールする​

以下の手順でQiskitをインストールします。

  1. Qiskitリポジトリをクローンします。
git clone https://github.com/Qiskit/qiskit.git
  1. qiskitディレクトリに移動します。
cd qiskit
  1. （オプション）テストまたはリントチェックを実行する場合は、開発者向け要件をインストールします。
pip install -r requirements-dev.txt
  1. qiskitをインストールします。

  • 標準インストール:

    pip install .

  • 編集可能モード: このモードでは、プロジェクトのコードが変更されてもQiskitを再インストールする必要はありません。

    pip install -e .

    編集可能モードでは、コンパイル済み拡張機能は最適化なしの_デバッグモード_でビルドされます。これはコンパイル済みコードの実行時パフォーマンスに影響します。最適化を有効にしてコンパイル済み拡張機能をビルドするには、次のコマンドを実行してバイナリを_リリースモード_で再ビルドしてください。

    python setup.py build_rust --release --inplace
    備考

    QiskitのRustコードを変更している場合、ローカルで変更を加えるたびに拡張コードを再ビルドする必要があります。編集可能モードでは、Rust拡張機能はインストールコマンドの実行時にのみビルドされるため、Rustコードへのローカルの変更は、build_rustを再実行して拡張機能を再ビルドしない限りインストール済みパッケージに反映されません（リリースモードとデバッグモードのどちらでビルドするかに応じて、--releaseの有無を選択してください）。

Qiskit Runtimeをインストールする​

Qiskit Runtimeをインストールする場合は、以下の手順に従ってください。

  1. Qiskit Runtimeリポジトリをクローンします。
git clone https://github.com/Qiskit/qiskit-ibm-runtime.git
  1. qiskit-ibm-runtimeディレクトリに移動します。
cd qiskit-ibm-runtime
  1. qiskit-runtimeをインストールします。グローバルなPythonインストールを汚染しないよう、仮想環境の使用を推奨します。

  • 標準インストール:

    pip install .

  • 編集可能モード: このモードでは、プロジェクトのコードが変更されてもQiskitを再インストールする必要はありません。

    pip install -e .

    編集可能モードでは、コンパイル済み拡張機能は最適化なしの_デバッグモード_でビルドされます。

  1. （オプション）テストまたはリントチェックを実行する場合は、開発者向け要件をインストールします。グローバルなPythonインストールを汚染しないよう、仮想環境の使用を推奨します。
pip install -e ".[dev]"

次のステップ​

推奨事項
  • コントリビューションガイドラインを読んで、オープンソースのQiskit SDKに貢献しましょう。
  • Hello worldプログラムを実行する。
  • Groverのアルゴリズムなどのチュートリアルを試してみましょう。
